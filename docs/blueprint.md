# **App Name**: Y-Ultimate Pulse

## Core Features:

- Home Page: Introductory page with mock login buttons for different user roles (Admin, Organizer, Participant).
- Dashboard: Role-based dashboard UI displaying relevant mock data based on the logged-in user's role.
- Event List: Display a list of mock events.
- Data Handling: Manage mock user, event, venue, and organization data using a single `mockData.ts` file.
- User Authentication Simulation: Simulate user authentication to identify the user's role and load mock data accordingly.
- Navigation: Implement simple navigation between the Home, Dashboard, and EventList pages.

## Style Guidelines:

- Primary color: Vibrant purple (#A020F0) to reflect energy and sophistication.
- Background color: Light gray (#F0F0F0), desaturated from the primary.
- Accent color: Soft blue (#7DF9FF), analogous to purple and adds a refreshing contrast.
- Body and headline font: 'Inter' for a modern and neutral feel.
- Use clear and consistent icons to represent different event types and actions.
- Implement a clean and responsive layout using TailwindCSS to ensure a consistent experience across devices.
- Subtle animations on page transitions and button interactions to improve user engagement.
